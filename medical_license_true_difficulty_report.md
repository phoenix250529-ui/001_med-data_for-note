# 医療系国家資格の「本当の合格率」分析レポート

## 概要

このレポートでは、医療系国家資格について、**国家試験の合格率だけでなく、受験資格取得のための選抜倍率と養成課程でのドロップアウト率を加味した「本当の合格率」**を算出しました。

7. **理学療法士**

## 各資格の入学難易度（偏差値）

### 医師（医学部医学科）
- **偏差値範囲**: 67～77
- **最低**: 67（地方国立医学部・私立医学部下位）
- **平均**: 70（標準的な国立医学部・私立医学部中堅）
- **最難関**: 77（東京大学理科三類）

**特徴**: 
- 全国82医学部、定員約9,400名
- 毎年約13万人が受験、倍率13.8倍
- 私立医学部は倍率26.2倍

### 歯科医師（歯学部歯学科）
- **偏差値範囲**: 50～65
- **平均**: 55

**特徴**: 医学部より入学難易度は低いが、それでも中堅以上の難易度

### 薬剤師（薬学部6年制）
- **偏差値範囲**: 44～65
- **最低**: 44（私立薬学部下位）
- **平均**: 57（中堅私立・国公立下位）
- **上位**: 65（国公立上位・慶應義塾大学）

**特徴**:
- 偏差値の幅が非常に広い
- **6年間ストレート卒業率: 67.3%**（留年率32.7%）
- 大学によっては退学率50%超

### 看護師（看護学部・看護専門学校）
- **偏差値範囲**: 40～60
- **専門学校**: 40～50（多くは45～50）
- **大学**: 48～60
- **平均**: 48

**特徴**:
- 養成機関が多様（大学・専門学校）
- 比較的入学しやすい

### 理学療法士・作業療法士
- **偏差値範囲**: 35～65
- **最低**: 35（専門学校下位）
- **平均**: 47～48
- **最難関**: 65（京都大学）

**特徴**:
- 養成校が急増（理学療法士279校、作業療法士約200校）
- 偏差値の幅が極めて広い

### 診療放射線技師・臨床検査技師
- **偏差値範囲**: 38～60
- **平均**: 50

|--------|------------------|--------------|--------------|
| 医師 | **84.9%** | 15.1% | 文部科学省2024年度 |
| 歯科医師 | **75%** (推定) | 25% | - |

| 作業療法士 | **86%** (推定) | 14% | - |

| 臨床検査技師 | **85%** (推定) | 15% | - |

**重要な発見**:
1. **医師は国家試験合格率は高いが、医学部での留年率も15.1%ある**
2. **薬剤師は留年率が32.7%と極めて高い**（10人に3人以上が留年）
3. 私立医学部は国公立より留年率が高い傾向（ストレート卒業率78.2% vs 83.2%）

## 「本当の合格率」の計算結果

### 計算式
```
本当の合格率 = 入学確率 × ストレート卒業率 × 新卒国試合格率
```

## 主要な発見

### 1. 医師の本当の難易度は圧倒的

- **国家試験合格率は92.4%と最高クラス**だが、
- **医学部入学の難易度（偏差値70 = 人口上位2.3%）**が極めて高いため、
- **本当の合格率は1.8%（54.5人に1人）**と、他の医療職と比べて圧倒的に低い

### 2. 薬剤師の意外な難しさ

- 国家試験合格率68.9%は低めだが、
- **さらに6年間で32.7%が留年する**ため、
- **本当の合格率は13.8%（7.2人に1人）**と、歯科医師よりも低い
- 入学は比較的容易でも、卒業と国試合格のハードルが高い

### 3. 看護師・理学療法士・作業療法士は比較的取得しやすい

- **本当の合格率45～47%**（約2人に1人）
- 入学のハードルが低く（偏差値47～48）、
- 養成課程での脱落率も比較的低い
- 国家試験合格率も高い

### 4. 国家試験合格率と本当の難易度は一致しない

| 資格 | 国試合格率ランク | 本当の難易度ランク | 逆転の理由 |
|------|------------------|-------------------|------------|
| 医師 | 1位（92.4%） | 8位（最難関） | 医学部入学が超難関 |
| 薬剤師 | 8位（68.9%） | 7位 | 高い留年率（32.7%） |
| 看護師 | 2位（90.1%） | 1位（最易） | 入学しやすい |

---

## 結論

### ショート動画で煽られる「国家試験合格率論争」の真実

**「医師国家試験の合格率は92%だから簡単」という主張は完全に誤り**

- 医師になれる確率（1.8%）は、看護師（47.2%）の約**26分の1**
- 薬剤師（13.8%）の約**7.5分の1**
- 理学療法士（47.2%）の約**26分の1**

**国家試験合格率だけで難易度を比較するのは無意味。入学選抜と養成課程でのフィルタリングを考慮しなければならない。**

### 各資格の特徴まとめ

#### 医師
- **超難関の入学選抜**（偏差値70 = 上位2.3%）
- 入学後も15.1%が留年
- 国試合格率は高い（95%）
- **総合難易度: 最高**

#### 薬剤師
- 入学は比較的容易（偏差値の幅が広い）
- **6年間で32.7%が留年**（最も高い）
- 国試合格率も低め（68.9%）
- **入ってからが地獄**

#### 看護師・理学療法士・作業療法士
- 入学しやすい（偏差値47～48）
- 養成課程でのフィルタリングは適度
- 国試合格率は高い
- **最も取得しやすい医療系国家資格**

---

## データの限界と注意点

1. **偏差値は養成機関によって大きく異なる**
   - 本分析では「平均的な偏差値」を使用
   - 実際には最低偏差値35から最高77までの幅がある

2. **ストレート卒業率は推定値を含む**
   - 医師・薬剤師は文部科学省の公式データ
   - その他は業界動向から推定

3. **地域差・施設差は考慮していない**
   - 都市部と地方での難易度差
   - 国公立と私立での差

4. **時代による変化**
   - 理学療法士・作業療法士の養成校は急増中
   - 今後さらに入学難易度が下がる可能性
